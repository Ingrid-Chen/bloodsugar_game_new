## 一、晨间 breakfast（AM 7:30）

### 1. 起晚了
**描述：** 起晚了，来不及好好吃饭。

| 选项 | 分值（血糖/心情/精力/饱腹） | 知识卡片 |
|------|----------------------------|----------|
| 当做轻断食，空腹扛着 | 0 / -5 / -25 / -20 | 过度断食会让皮质醇空转导致严重脑雾，隔夜空腹叠加持续禁食使大脑葡萄糖供应不足，中午极易因「补偿心理」暴食。 |
| 路边买根热甜玉米边走边啃 | +20 / +10 / +15 / +20 | 甜玉米含有较多直链淀粉和膳食纤维，升糖速度比精制谷物慢，能有效阻止精力崩盘。 |

## 附：游戏规则摘要

- **死亡判定：** 血糖 > 100（高渗昏迷）、血糖 ≤ 10（低血糖休克）、精力 ≤ 0（过劳晕倒）、心情 ≤ 0（精神崩溃）。
- **饱腹惩罚：** 饱腹 ≥ 100 时触发「撑得大脑缺氧」：饱腹→90，精力-15，心情-10；饱腹 ≤ 0 时触发「饿得眼冒金星」：饱腹→10，血糖-10，心情-10。
- **时段限制：** 标「仅周末」的事件周六、周日出现；标「仅工作日」的事件周一至周五出现。
- **周一晨间：** 固定为事件 1「起晚了」，其余时段从对应池中随机抽取（下午/深夜各 50% 概率出现）。
